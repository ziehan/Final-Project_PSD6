# Final-Project_PSD6

## Game of Life

Sebuah hardware yang mensimulasikan kehidupan sel dalam sebuah papan grid. Setiap kotak (sel) punya "otak" sendiri untuk menentukan apakah dia hidup atau mati berdasarkan kondisi tetangganya. Semua sel menghitung nasibnya secara bersamaan (paralel), bukan satu-satu. Dilengkapi CPU kecil untuk menggambar pola awal sebelum simulasi jalan.
